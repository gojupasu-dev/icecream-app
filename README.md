# 무인아이스크림 사장님 운영 대시보드 v6

이 패키지는 GitHub Pages에 바로 업로드할 수 있는 정적 사이트입니다.

## 중요한 점
현재 미리보기에서 실제로 입력해둔 데이터는 브라우저 localStorage에 저장되어 있어서,
이 ZIP 안에 자동 포함되지는 않습니다.

### 데이터 옮기는 방법
1. 기존 앱에서 `전체 백업` 버튼 클릭
2. JSON 파일 저장
3. 이 v6 버전을 GitHub에 업로드
4. 새 사이트에서 `백업 불러오기`로 JSON 선택

## 배포 방법
1. 이 ZIP 압축 해제
2. GitHub 저장소 열기
3. Add file → Upload files
4. `index.html`, `.nojekyll`, `README.md` 업로드
5. Commit changes
6. GitHub Pages가 켜져 있으면 몇 분 뒤 반영

## 이번 버전 특징
- POS형 직관성 + 분석 대시보드형 카드 레이아웃
- 더 큰 카드, 더 선명한 핵심 숫자
- 파랑/청록/오렌지 중심 색상 구성
- 탭과 버튼 가시성 개선
- 기존 기능 유지
